🚀 PASOS PARA EMPEZAR (En orden)
═══════════════════════════════════════════════════════════════════════════════

1️⃣  LEE CHECKLIST.md (5 min) - Entiende qué hay que hacer

2️⃣  GOOGLE APPS SCRIPT (10 min)
    - Abre tu Google Sheet
    - Editor de secuencias → Pega Code.gs
    - Desplega como "Ejecución web"
    - Copia el ID (IMPORTANTE!)

3️⃣  GITHUB (10 min)
    - Crea un repo llamado "agenda-inteligente"
    - Sube los archivos: git push

4️⃣  CLOUDFLARE PAGES (5 min)
    - Conecta tu repo de GitHub
    - Auto-deploy a: https://agenda-inteligente.pages.dev

5️⃣  CONFIGURAR LA APP (5 min)
    - Abre tu URL de Cloudflare
    - Ingresa tu email Gmail
    - Ingresa el ID del Apps Script
    - Activa los triggers automáticos

6️⃣  PROBAR (10 min)
    - Crea un evento de prueba
    - Verifica que sincronice con Sheets
    - Revisa tu email para notificación
    - Instala en tu móvil
